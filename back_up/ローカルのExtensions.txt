ここは「Availableタブ→Load from:ボタン」で入れられるやつのページ

1111に追加インストールする拡張機能やでー
使った人は感想とか書いてって欲しい
「Install from URLタブ」で入れるやつは[[Extensions_URL]]に書こう

#contents

* 導入方法

インストールがwebで完結するので簡単

+ 1111が古かったら新しくする
+ --listenか--shareしてるなら[[ローカルのwebui-user.bat]]も見て
+ Extensionsタブ→Availableタブ→Load from:ボタン
+ 好きなのを選んでInstall

** 使い方

だいたいは、タブが増えてるか、txt2img/img2imgのScriptドロップダウンで選ぶか。
どっちにも無かったらreadmeとかちゃんと読んだほうがいい。

*** リポジトリへのリンク

AvailableタブのExtension名のリンクから配布元リポジトリに飛べる
readmeもそこにあるはず

** アップデート方法

InstalledタブのCheck for updates→Applyを押す

** 注意

- インストーラーがついててvenvに何か入れるものも多い
- 安定とは限らない
-- アンインストールはフォルダごと消すほうが安全っぽい

** Prompt from file or textbox

- 複数のプロンプトで生成する
- "A","B"と入れると、AのプロンプトとBのプロンプトでそれぞれ出力する
- 複数行テキストを入力すると1行1プロンプトで出力
- 以下のような形式でプロンプト以外も指定可能
=|BOX|
--prompt "プロンプトA" --negative_prompt "ネガティブプロンプトA" --outpath_samples "outputs/txt2img-images/A" --batch_size 2 --n_iter 5 --sampler_name "Euler a" --cfg_scale 7 --steps 25 --seed 200000000
--prompt "プロンプトB" --negative_prompt "ネガティブプロンプトB" --outpath_samples "outputs/txt2img-images/B" --batch_size 2 --n_iter 2 --sampler_name "Euler" --cfg_scale 10 --steps 30 --seed 200000000
||=
-- 指定可能な要素
--- --sd_model: model (get_closet_checkpoint_matchで検索するための文字列、モデル名の一部分など)
--- --outpath_samples: string
--- --outpath_grids: string
--- --prompt_for_display: string (本体側が使用していない変数のため指定する意味が無い)
--- --prompt: string
--- --negative_prompt: string
--- --styles: string
--- --seed: int
--- --subseed_strength: float
--- --subseed: int
--- --seed_resize_from_h: int
--- --seed_resize_from_w: int
--- --sampler_index: int (本体側が使用していない変数のため指定する意味が無い)
--- --sampler_name: string
--- --batch_size: int
--- --n_iter: int
--- --steps: int
--- --cfg_scale: int
--- --width: int
--- --height: int
--- --restore_faces: boolean
--- --tiling: boolean
--- --do_not_save_samples: boolean
--- --do_not_save_grid: boolean
